# CyberQuest - Gamified Cybersecurity Learning Platform

CyberQuest is a gamified cybersecurity learning platform where students can solve security challenges, earn points, and track their progress through leaderboards and achievements.

The platform makes cybersecurity learning interactive and engaging.
3. Features

Example:

User Authentication (JWT Login & Register)

Interactive Cybersecurity Quiz Game

Leaderboard (Global & Weekly)

AI Mentor Chat Assistant

Student Profile Dashboard

Gamification (Points, Badges, Levels)
